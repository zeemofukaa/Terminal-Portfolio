### (˶ᵔ ᵕ ᵔ˶) Terminal Portfolio

A portfolio that runs entirely inside the terminal.

Built with Go and Bubble Tea, it combines a terminal user interface with interactive navigation, animated effects, and a few hidden surprises.

-------------------------------------------------------------------------------------
✦ **Features**

* Terminal-based user interface
* Keyboard-driven navigation
* Interactive project showcase
* Skills and contact sections
* Hidden easter egg
* Lightweight, cross-platform application

-------------------------------------------------------------------------------------
✦ **Built With**

* Go
* Bubble Tea
* Lip Gloss
* Bubbles

-------------------------------------------------------------------------------------
✦ **Requirements**

Before running the project, make sure you have the following installed:

* **Go** (1.24 or later): https://go.dev/dl/
* **Git**: https://git-scm.com/downloads

Verify the installation:

```bash
go version
git --version
```

-------------------------------------------------------------------------------------
✦ **Running Locally**

Clone the repository:

```bash
git clone https://github.com/zeemofukaa/Terminal-Portfolio.git
cd Terminal-Portfolio
```

Run the application:

```bash
go run .
```

-------------------------------------------------------------------------------------
✦ **Project Structure**

```text
internal/
 ├── sections/    -> individual page components
 └── ui/          -> application layout and navigation

main.go           -> application entry point
go.mod            -> module definition
go.sum            -> dependency checksums
```
